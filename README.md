## NodeJS lambda starter template

### requirements
```
   node > 12.x
   serverless framework
   typescript
```

### how to use 
```
   clone repository 
   cd nodejs-typescript-lamda-template
   npm i 
   npm run run-local
```

### test local
With server runs locally

```
  curl -XGET -H "Content-type: application/json" 'http://localhost:3000/health'
```
